Формат файлов.

.DOC или .doc (от англ. document) - расширение имени файла, используемое для файлов, представляющих текст, с разметкой или без. Расширение .DOC часто использовалось для обозначение простых текстовых файлов без формирования, однако позже стало использоваться для двоичных форматов с разметкой. Документ MS Word
Portable Document Format (PDF) - межплатформенный формат электронных документов, разработанный фирмой Adobe Systems с использованием ряда возможностей языка Postscript. Документ PDF
ZIP - популярныый формат сжатия данных и архивации файлов. Файл в этом формате обычно имеет раширение .zip и хранит в сжатом или несжатом виде один или несколько файлов, которое можно из него извлечь путем распаковки с помощью специальной программы. Архив
