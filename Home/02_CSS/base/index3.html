Размер блок: Занммает всю доступную ширину Должен присутствоватьцвет фона Во всех "статях" сделайте отступ снизу от соседей равным .2em, при помощи свойства margin- bottom. Картинка вставлена при помощи тега, плавание организовано по правлму краю внутри данной статьи для этой картинки.
